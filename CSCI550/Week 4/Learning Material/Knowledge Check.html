Question 1: Which design pattern is best suited for dynamically selecting secure algorithms or communication protocols based on context?

Correct answer: b. Strategy Pattern Explanation: The Strategy pattern allows dynamic selection of secure algorithms or protocols, enhancing flexibility while maintaining security.

Scenario: During a security review of a codebase, your team discovers that sensitive API keys are hardcoded in the source code. What are two recommended actions to resolve this issue and prevent it in the future?

Use a secure secret management tool (e.g., HashiCorp Vault or AWS Secrets Manager) to store and retrieve sensitive information securely.

  1. Use a secure secret management tool (e.g., HashiCorp Vault or AWS Secrets Manager) to store and retrieve sensitive information securely.
  2. Refactor the code to retrieve the API keys from environment variables or secure configurations instead of hardcoding them directly.

Explanation: Hardcoding sensitive information is a common anti-pattern. Using secure tools and practices helps reduce the risk of accidental exposure or unauthorized access.
