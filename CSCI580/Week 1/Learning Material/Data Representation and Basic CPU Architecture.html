Data Formats

Data formats define how data is structured and stored in computer systems. Key data formats include the following:

Integer Representation:

  • Fixed-point representation for whole numbers.
  • Can be signed (positive and negative) or unsigned (only positive).
  • Common sizes are 8-bit, 16-bit, 32-bit, and 64-bit format.

Floating-Point Representation:

  • Used for representing real numbers with fractional parts.
  • Follows the IEEE 754 standard, which defines formats for 32-bit (single-precision) and 64-bit (double-precision) floating-point numbers.
  • Consists of sign, exponent, and mantissa parts.

Binary Coded Decimal (BCD):

  • Represents each decimal digit with its four-bit binary equivalent.
  • Used in applications requiring precise decimal representation, such as financial calculations.

ASCII and Unicode:

  • Character encoding standards that map characters to numerical values.
  • ASCII uses seven or eight bits per character, supporting 128 or 256 characters.
  • Unicode supports a wider range of characters with variable-length encoding, allowing representation of characters from virtually all writing systems.

Basic CPU Components

The CPU is the brain of a computer system, responsible for executing instructions and controlling the operation of other components. Understanding the basic components of a CPU is crucial for grasping how computers process information. Let‚Äôs explore the key elements that make up a modern CPU.


  • The ALU is responsible for performing all arithmetic and logical operations.
  • It handles basic calculations like addition, subtraction, multiplication, and division.
  • Logical operations such as AND, OR, NOT, and XOR are also performed by the ALU.
  • Modern CPUs often contain multiple ALUs to handle parallel processing tasks, enhancing computational efficiency.
  • The control unit manages the execution of instructions by directing the operation of the other components of the CPU.
  • It fetches instructions from memory, decodes them, and then executes them by coordinating with the ALU and other units.
  • In modern CPUs, the control unit is integrated with advanced features like pipelining and out-of-order execution to improve instruction throughput.
  • Registers are small, fast storage locations within the CPU used to hold temporary data and instructions that are being processed.
  • They play a crucial role in the execution of instructions by providing quick access to data.
  • Modern CPUs have a variety of registers, including the following types:
    • General-purpose registers: Used for temporary data storage during computations.
    • Instruction registers: Hold the current instruction being executed.
    • Program counter: Keeps track of the address of the next instruction to be executed.
    • Stack pointer: Points to the top of the stack in memory.
  • Cache memory is a small-sized type of volatile computer memory that provides high-speed data storage and access to the CPU.
  • It stores frequently accessed data and instructions, reducing the time needed to fetch them from the main memory.
  • Modern CPUs have multiple levels of cache (L1, L2, and sometimes L3) to optimize data retrieval speed.
  • These caches use sophisticated algorithms to predict and pre-load data that the CPU is likely to need next.
  • The MMU handles the translation of virtual memory addresses to physical addresses, enabling efficient memory management and protection.
  • It supports advanced features like address space layout randomization (ASLR) to enhance security.
  • The MMU is crucial for supporting large address spaces in 64-bit architectures.
